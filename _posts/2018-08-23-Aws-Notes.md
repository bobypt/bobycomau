# Useful Commands  

## SSH to an EC2 instnace

```
ssh -i ~/repos/aws/keypair/awsAccess.pem ec2-user@52.65.42.180
```

## Copy a file to remote server

```
scp -i awsAccess.pem awsAccess.pem ec2-user@52.65.42.180:/tmp/
```
